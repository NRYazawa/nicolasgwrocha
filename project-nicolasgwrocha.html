Project Overview

This website was conceptualized and built not just as a static resume, but as a standalone development project. It marks my first comprehensive venture into front-end web development, aimed at providing a clean, responsive, and professional environment to showcase my software engineering portfolio.


Key Technical Implementations

1. Custom Dark/Light Mode Architecture

I implemented a robust CSS architecture capable of detecting user system preferences to automatically switch between custom light and dark themes. This required writing absolute CSS overrides, managing media queries, and ensuring high-contrast readability across both viewing modes.

2. Responsive Grid System

The site utilizes a customized responsive grid, adapting fluidly from large desktop monitors down to mobile devices. I re-engineered legacy template constraints to create horizontal card layouts and dynamic image scaling, proving my ability to manage and debug front-end rendering logic.
