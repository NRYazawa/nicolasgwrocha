Twine HTML / CSS Branching Narrative State Management
  • Play on Itch.io

Project Overview

Blackheart_ is a text-based interactive fiction game developed entirely in Twine using HTML and CSS. You are hired as an IT Technician to eradicate a virus from the computer systems of a high-security Soviet penitentiary. However, as you dig deeper into the system, you quickly realize that nothing is as it seems, and you cannot trust anyone.


Key Technical Implementations

1. Branching Logic & State Tracking

Despite being a text-based experience, the game relies heavily on underlying programming logic to track player choices, inventory states, and narrative consequences. Managing these variables without a traditional game engine was an excellent exercise in fundamental programming logic and state machine structuring.

2. Immersive UI Formatting

To sell the illusion of operating an old Soviet computer terminal, I utilized custom HTML and CSS formatting. The visual design of the text, background contrasts, and hyperlinking flow were all carefully engineered to keep the player immersed in the role of a technician navigating a compromised operating system.
